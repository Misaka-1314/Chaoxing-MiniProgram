网页入口用于小程序被举报封号时备用，部分功能不可用！获取签到由 [小北云](https://github.com/xunixianshituandui/xiaobei) 提供接口支持！

:::

:::warning
请在手机端点击去签到按钮，电脑点击无法使用！
:::
