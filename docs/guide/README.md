## 关于我们

#### 1. 这是什么？

御坂学习小程序 是一款学习通签到小程序，帮助学有余力的同学完成签到任务，提升自学效率。

#### 2. 什么情况下使用？

本小程序仅适用于教室网络过差无法正常签到、学习过程中不希望被签到打断学习节奏等场景下使用。禁止以逃课、旷课等行为作为使用目的，由用户非法行为造成的后果均由用户自行负责。

#### 3. 包括哪些功能？

[点击查看支持的签到类型](./type.md)，支持各种类型的签到，但不支持实习打卡、抢答等功能。
